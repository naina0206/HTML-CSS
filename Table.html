Table tag- Used to define table in html. Used to display and format tabular data.

It has three tags- tr, td and th.
1. tr- Used to display table row.
2. td- for table data.
3. th- Used in place for table data for displaying table headers.

We can also add caption in table by caption tag.

There are two more tags- thead, tbody.
thead-Acts as a container for header and caption.
tbody- Acts as a container for rest of the rows.
